# مشروع المحفظة الشخصية - مهندس البرمجيات عز الدين خالد عبيد

## 🎯 نظرة عامة
موقع محفظة شخصية تفاعلية ثنائية اللغة (عربي/إنجليزي) تعرض خبرات ومهارات مهندس برمجيات وأخصائي ذكاء اصطناعي، مع لوحة تحكم CMS لحظية ونظام تخزين محلي.

## 🏗️ التقنيات المستخدمة
- **HTML5** - هيكل الصفحات
- **CSS3** - جميع الأنماط (بدون إطار عمل)
- **JavaScript (Vanilla ES6+)** - المنطق والتفاعل
- **LocalStorage** - تخزين البيانات والمزامنة
- **CSS Custom Properties** - نظام الألوان والثيمات
- **Intersection Observer** - تأثيرات الظهور المتحرك

## 🔄 تدفق البيانات

```
INITIAL_DATA (data.js)
      ↓
localStorage.load() ← أول تحميل / revert
      ↓
App State (كائن JavaScript مركزي)
      ↓
renderAll() ← تعيد بناء جميع أقسام الواجهة
      ↓
DOM ← يتفاعل مع المستخدم
      ↓
CMS Dashboard ← يعدل الـ State
      ↓
localStorage.save() ← حفظ تلقائي
      ↓
Export/Import JSON ← تصدير/استيراد يدوي
```

## 🧩 المكونات الرئيسية

### 1. شريط التنقل (Navbar)
- ثابت زجاجي (glassmorphic sticky)
- تبديل لغة عربي/إنجليزي
- أزرار للأقسام الرئيسية
- زر فتح لوحة CMS

### 2. قسم الـ Hero
- اسم ثنائي اللغة
- مسمى وظيفي
- وصف فرعي
- 4 مؤشرات أداء رقمية متحركة

### 3. قسم السيرة الذاتية (About)
- سيرة تفصيلية
- المؤهل الأكاديمي
- مجال التركيز التقني

### 4. ترسانة المهارات (Tech Stack)
- 4 تصنيفات: AI, Languages, Backend, Tools
- أشرطة تقدم نسبية
- فلترة بالتصنيف والبحث

### 5. معرض المشاريع (Projects)
- 3 بطاقات مشاريع مع توهجات ملونة
- hover effect مع glow
- روابط GitHub و Demo

### 6. الجدول الزمني (Experience)
- خط زمني عمودي
- شركتين مع وصف المهام

### 7. خارطة الطريق (Roadmap)
- 3 أهداف بحثية
- حالة كل هدف (Active/Learning/Research)

### 8. قسم التواصل (Contact)
- معلومات الاتصال
- نموذج إرسال رسائل
- صندوق وارد محاكي (Simulated Inbox)

### 9. لوحة CMS (Drawer)
- تحرير جميع الأقسام
- إضافة/حذف مهارات ومشاريع
- تصدير/استيراد JSON
- استعادة الإعدادات الافتراضية
